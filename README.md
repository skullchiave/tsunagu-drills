# つなぐにほんご 動詞・形容詞ドリル

日本語学校の授業用ドリル集。GitHub Pages で公開し、教室のプロジェクター投影と学生のスマホ両方で使う。

公開URL（学生に配る入口）: https://skullchiave.github.io/tsunagu-drills/

## 使い方

- ルートの `index.html` が一覧トップ（母艦）。ここから各ドリルへ進む
- 各 HTML が 1 ドリル。URL を開くだけで動く（インストール・ログイン不要）
- 1 つの HTML がレスポンシブ対応：PC/プロジェクターでは投影用レイアウト、スマホ（幅 640px 以下）ではタップ操作用レイアウトに自動で切り替わる

## 編集・公開の手順

1. ローカルのリポジトリで HTML を編集
2. ブラウザで開いて動作確認（ウィンドウを狭めるとスマホ用レイアウトを確認できる）
3. `git add` → `git commit` → `git push` で公開に反映（数十秒〜1分で URL に反映）

## フォルダ構成

- ルート直下の `*.html` … 公開されるドリル本体
- `_original/` … 原本のコピー（git 追跡しない・参照用）
- `scratch/`, `tmp/` … 実験・一時ファイル（git 追跡しない）
