「なかった」（過去否定かこひていの 普通形ふつうけい）の つくりかた

ない形けい（7-2）の 「ない」を「なかった」に かえるだけ。「ませんでした」の カジュアルな いいかた。

1グループ

「ません」の まえ（い段だん）を あ段だんに かえて ＋なかった
いきません → いかなかった
のみません → のまなかった
かきません → かかなかった
はなしません → はなさなかった

2グループ

「ません」を とって ＋なかった
たべません → たべなかった
みません → みなかった
あけません → あけなかった
おきません → おきなかった

3グループ

2つだけ（おぼえる）
きません → こなかった
しません → しなかった
べんきょうしません →
べんきょうしなかった
ちゅうい： 「い」は あ段だんで わ に なる：かいません → かわなかった（×かあなかった）
ありません の 過去かこは なかった（×あらなかった）。
ない形けいを しっていたら かんたん：いかない → いかなかった ／ こない → こなかった（ない → なかった）。
