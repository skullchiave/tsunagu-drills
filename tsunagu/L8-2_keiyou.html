形容詞けいようし・名詞めいしの 過去かこ（カジュアル）

きのうの ことを ともだちと はなす。「〜です」を 過去かこの 普通形ふつうけいに かえる。

い形容詞けいようし

〜い → 〜かった
〜い → 〜くなかった
おもしろいです → おもしろかった
おもしろいです → おもしろくなかった
むずかしいです → むずかしかった
さむいです → さむくなかった

な形容詞けいようし

〜です → 〜だった
〜です → 〜じゃなかった
かんたんです → かんたんだった
かんたんです → かんたんじゃなかった
ひまです → ひまだった
にぎやかです → にぎやかじゃなかった

名詞めいし

な形容詞けいようしと おなじ
〜だった ／ 〜じゃなかった
テストです → テストだった
テストです → テストじゃなかった
やすみです → やすみだった
こどもです → こどもじゃなかった
ちゅうい： いい は とくべつ → よかった ／ よくなかった（×いかった ×いくなかった）。
きれい・ゆうめい は「い」で おわるけど な形容詞けいようし → きれいだった ／ きれいじゃなかった。
